Cortexa — Beyond the Imagination

Project Description

That is a powerhouse of a tech stack! Since Cortexa AI covers a range of complex behaviors—emotions, alertness (drowsiness), and spatial orientation (head pose).

Cortexa AI is an advanced computer vision system designed to quantify human behavior through real-time facial analysis. By integrating Deep Learning models via TensorFlow, the platform simultaneously tracks emotional states, monitors fatigue levels through drowsiness detection, and calculates 3D head orientation.

Key Features

  1. Multi-Dimensional Behavioral Analytics
    • Real-time Emotion Mapping: Identifies micro-expressions to gauge sentiment and mental well-being.
    • Predictive Drowsiness Alerts: Uses Eye Aspect Ratio (EAR) and blink frequency to provide early warnings for fatigue—critical for drivers or night-shift workers.
    • 3D Head Pose Estimation: Tracks spatial orientation (Pitch, Yaw, and Roll) to monitor attention levels and ergonomic posture.
  2. Behavioral Dashboards: Interactive charts that visualize emotional trends over time.
  3. Personalized Calibration: A smart setup wizard that helps the AI learn the user's neutral face and "resting" pose for higher accuracy.
  4. Universal Cross-Platform Architecture
    • Write Once, Deploy Anywhere: Built with a decoupled Flask-CORS backend, allowing the same logic to power a Chrome Extension (for productivity), a Mobile App (for driving safety), or a Web Portal (for remote learning).
    • Lightweight Edge Processing: Optimized for low latency, ensuring the video stream is processed efficiently regardless of the device hardware.

How to Run Locally

Prerequisites: Make sure you have Python (3.7+)

1. Setup

git clone https://github.com/abinavgt/Cortexa.git
cd ./cortexai-main

2. Activate your Virtual Environment

python -m venv myenv --without-pip
myenv\Scripts\python -m ensurepip --upgrade
myenv\Scripts\activate

3. Install Dependencies

pip install -r requirements.txt

4. Run the Application

python app.py

Outcomes

  1. Synchronous Inference: Successfully achieved real-time, simultaneous execution of three deep learning models (Emotion, Drowsiness, and Head Pose) without significant frame-rate drops.
  2. Precision Accuracy: Reliable detection of micro-expressions and eye closure patterns, providing a high degree of confidence in the generated behavioral data.
  3. Visual Trend Analysis: Users can identify specific times of day when their focus drops or their stress (emotions) peaks through the integrated dashboard.
  4. Universal Compatibility: The system is optimized to run across different hardware environments, from standard PC webcams to mobile camera modules.
